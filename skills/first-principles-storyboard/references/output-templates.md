# 输出格式模板

## 分镜表

```markdown
| 镜号 | 景别 | 视角角度 | 镜头运动 | 时长 | 画面内容 | 声音 | 核心目标 |
|------|------|---------|---------|------|---------|------|---------|
| 1 | 全景 | 平视 | 固定 | 3s | ... | ... | ... |
```

## 创作说明

```markdown
**段落功能**：一句话说明段落戏剧任务
**情绪曲线**：描述情绪起伏
**视角策略**：说明视角选择原因
**关键转折点**：标记情绪转折位置
```

## 修改说明（第N轮）

```markdown
| 扣分点 | 原问题 | 修改内容 | 修改原因 |
|--------|--------|---------|---------|
```

## AI生图提示词

```markdown
### 镜头1
[景别], [角度], [光影], [色彩], [画面描述], 电影质感, 写实风格
```
