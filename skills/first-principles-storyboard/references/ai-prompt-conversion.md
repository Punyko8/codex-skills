# AI 生图提示词自动转换规则

## 标准格式

`[景别], [拍摄角度], [光影], [色彩], [画面内容], [风格]`

## 禁止出现

慢动作、手持抖动、推/拉/摇/移、情绪形容词、抽象概念

## 情绪→视觉元素转化表

| 情绪 | 视觉转化 |
|------|----------|
| 紧张 | 昏暗光影、高对比度、倾斜构图 |
| 悲伤 | 冷色调、柔和漫射光、空旷构图 |
| 不安 | 轻微模糊/晃动感、低饱和度 |
| 希望 | 暖色调、柔光、明亮背景 |

## 示例

**分镜描述**：大特写，平视，固定，3s，透明玻璃杯的杯口，一缕白色的热气在暖黄色的台灯光下缓缓升起

**AI生图提示词**：大特写，平视角度，暖黄色台灯光影，低对比度，透明玻璃杯的杯口，一缕白色的热气缓缓升起，电影质感，写实风格
