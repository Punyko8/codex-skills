# 镜号分法实操指南

## 为什么需要这个指南

写分镜时常见的返工原因之一是**镜号分法不合理**——镜号划得太碎或太粗，导致导演/甲方反复调整顺序，浪费迭代轮次。

本指南提供两条核心规则：**对话对齐规则**和**动作对齐规则**。任何镜号分法问题，都能归到其中一条。

---

## 规则一：对话对齐规则

> **一句话：一个问答对 = 一组镜号，问句和答句不能在同一个镜号里各占一半。**

### 正确做法

把一段对话按「谁问→谁答」的自然对切拆成镜号。

```
❌ 错误（把两个问答对塞进同两个镜号）：
镜 10 | 中景 | 妈妈问"想去看吗？"爸爸问"为什么？"小吉答"不啦""拿着气球不能玩滑梯"

✅ 正确（每个问答对独立成号）：
镜 10  | 妈妈问"想去看吗？" → 小吉答"不啦"
镜 10.1| 爸爸问"为什么？" → 小吉答"拿着气球，就不能玩滑梯啦"
```

### 判断标准

- 观众在同一个镜头里需要接收**一个问句+其对应的答句** → 独立镜号
- 一个镜头里出现了**两个不同的说话人发起对话** → 必须拆分

### 例外

- 群戏/多人快速插话：可以合并，但不超过3个话轮
- 叠话（两人同时说同一句）：不用拆分

---

## 规则二：动作对齐规则

> **一句话：观众看到的动作阶段决定镜号边界，不是剧本里的句子边界。**

### 正确做法

把一段连续动作按「观众能感知的动作阶段」拆分。

```
❌ 错误（动作阶段混在一起）：
镜 21 | 小吉松开线，气球慢慢上飘

✅ 正确（拆分出独立的视觉阶段）：
镜 21  | 小吉松开线（手部动作）
镜 21.1| 气球慢慢往上飘（观众单独感知飘起的瞬间）
```

### 判断标准

问：观众需要单独处理这个动作的「开始」「过程」「结果」吗？
- 只是松线手部特写 → 独立一镜
- 「气球开始飘」需要单独给时间让观众感受 → 独立一镜
- 结果/反应（小吉表情变化） → 独立一镜

### 三条子规则

1. **触发动作为一镜**：动作的起手/触发动作，给它一个特写
2. **效果动作为一镜**：触发后的视觉结果，给时间让观众看
3. **反应用一镜**：角色对上一动作的反应，单独一镜

---

## 规则三：动作逻辑决定顺序

> **一句话：观众先看到「做什么」，再看到「为什么这样做 / 模仿这样做」。**

### 应用场景

当多个角色做类似动作时（A做 → B模仿 / A摔跤 → B取笑）：

```
镜 20  | 皮皮绑线缠住摔跤
镜 20.1| 小吉学着滚来滚去
```

先展示"发生了什么"（摔跤），再展示"别人怎么反应"（滚来滚去取乐）。如果顺序反了，观众先看到模仿，再回头看被模仿的动作——观感混乱。

### 结论

> **动作逻辑链 = 触发 → 效果 → 反应 → 模仿/互动**
>
> 每一步都可能独立成镜，按此顺序编排，不要直觉排序。
