# 四色调度箭头规则

## 画面风格

故事板主体必须是黑白手绘线稿。彩色只用于调度箭头。这样可以避免图像模型把画面内容和调度标注混在一起。

## 固定颜色

| 颜色 | 含义 | 示例 |
|---|---|---|
| 红色箭头 | 角色主要运动 | 主角从左走到右、伸手拿东西、跑向朋友 |
| 蓝色箭头 | 镜头运动 | 推近、拉远、横移、摇镜 |
| 绿色箭头 | 视线/注意力方向 | 主角看向掉落玩具、朋友看向主角 |
| 黄色箭头 | 关键物体运动或变化 | 球滚动、积木倒下、牙刷被拿起 |

## 数量限制

每格最多 3 条箭头。优先级：

1. 儿童理解因果必须看见的运动。
2. 关键视线方向。
3. 镜头运动。
4. 次要物体运动。

如果超过 3 条，删除不影响因果理解的箭头；必要时建议拆镜头或拆故事板。

## 箭头描述格式

每格写清：

- 箭头颜色。
- 起点和终点。
- 标注对象。
- 对儿童理解的作用。

示例：

```text
格 2：红色箭头从主角身体指向玩具箱，表示主角走向玩具箱；绿色箭头从主角眼睛指向地上的积木，表示主角看见问题。
```

## 镜头运动表达

蓝色箭头只表示镜头调度，不表示角色移动。

- 推近：蓝色箭头从画面边缘指向主体。
- 拉远：蓝色箭头从主体指向画面外侧。
- 横移：蓝色箭头沿画面左/右方向横向。
- 摇镜：蓝色弧形箭头标注视野转向。
- 定镜：不要画蓝色箭头，可在图外标注“定镜”。

## 禁止做法

不要把箭头画成装饰线条。不要让箭头穿过角色脸部、手部和关键道具。不要用彩色填充角色或背景。不要在格内放大段文字。
