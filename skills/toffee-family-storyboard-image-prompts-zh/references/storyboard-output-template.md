# 故事板提示词固定输出模板

用户确认分段方案后，每张故事板都按以下结构输出。

```markdown
## SB-01｜段落标题

包含镜头：S01-S03
宫格类型：单张 / 二宫格 / 四宫格
段落目的：一句话说明这一段让儿童看懂什么。
宫格选择理由：说明为什么用单张、二宫格或四宫格。

设定图使用说明：
已参考主角设定图和道具图。故事板提示词不展开角色外观，仅使用“主角”“朋友”“关键道具”等制作代称。若后续图像生成，需要将设定图作为参考图一并输入。

图外信息区：
- 顶部信息：故事板编号、段落标题、包含镜头号、宫格类型。
- 图例：红色箭头 = 角色主要运动；蓝色箭头 = 镜头运动；绿色箭头 = 视线/注意力方向；黄色箭头 = 关键物体运动或变化。
- 格外标注：
  - 格 1 / 对应镜头 S01：简述格外标注。
  - 格 2 / 对应镜头 S02：简述格外标注。
- 底部检查：儿童理解功能与风险提醒。

故事板总提示词：
生成一张【单张/二宫格/四宫格】儿童动画制作故事板。主体画面为黑白手绘线稿风格，线条清楚，背景简化，只保留理解故事所需场景信息。每格画面外侧保留信息区，格内不要放大段中文文字。只使用彩色箭头表达调度：红色箭头表示角色主要运动，蓝色箭头表示镜头运动，绿色箭头表示视线/注意力方向，黄色箭头表示关键物体运动或变化。每格最多 3 条箭头。画面重点是镜头调度、人物运动、视线方向和关键物体变化，不要画成精美成片海报，不要复杂光影，不要改变指定宫格数量。

逐格提示词：
- 格 1｜对应镜头 S01：黑白手绘线稿，描述景别、构图、主体位置、角色动作、表情、视线、关键道具、需要出现的箭头。
- 格 2｜对应镜头 S02：黑白手绘线稿，描述景别、构图、主体位置、角色动作、表情、视线、关键道具、需要出现的箭头。

调度箭头说明：
- 格 1：红色箭头……；绿色箭头……；蓝色箭头……
- 格 2：黄色箭头……；绿色箭头……

低幼可理解性自检：
- 通过：列出主角、问题、动作变化、情绪原因、箭头辅助是否清楚。
- 需调整：如无问题写“无”；如有问题，写具体镜头号和修改建议。
```

## 设定图不确定时

将“设定图使用说明”替换为：

```markdown
设定图需确认：
主角的常用服装在设定图 A 与 B 中不一致，请确认故事板阶段使用哪一版。
```

## 多镜头对应一格

如一格对应多个镜头，必须写清原因：

```markdown
- 格 2｜对应镜头 S04-S05：两个镜头属于同一动作结果，画面表现为主角伸手后够不到道具；图外标注保留 S04-S05。
```

## 调整提醒格式

如果发现原镜头有问题，在故事板末尾追加：

```markdown
原镜头调整建议：
- S06：同时包含“跑过去、摔倒、哭、朋友看见”，信息过密。建议拆成 S06a 跑过去，S06b 摔倒后的表情反应。
```
