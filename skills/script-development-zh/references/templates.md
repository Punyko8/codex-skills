# 剧本开发输出模板

Use these templates when `script-development-zh` reaches the corresponding workflow stage. Keep outputs in Chinese unless the user explicitly asks otherwise.

## DNA 提取输出格式

```text
《[作品名]》创作 DNA 提取
──────────────────────────────────

规律 1：[一句话概括]  【置信度：高】
   观察依据：____
   对本次创作的指导意义：____

规律 2：[一句话概括]  【置信度：中】
   观察依据：____
   对本次创作的指导意义：____

规律 3：[一句话概括]  【置信度：低】
   观察依据：____
   对本次创作的指导意义：____

（共 N 条，N ∈ [5, 7]）

请审视上述规律：
  • 哪几条你认同？（如“保留 1、3、5”）
  • 哪几条要修改？
  • 哪几条不适用本次创作？
  • 你想补充哪几条本作品的特点？
```

## Story Synopsis

Use for 30 seconds to 5 minutes.

```text
开端：____（钩子 + 角色登场）
冲突：____（核心矛盾爆发）
收尾：____（情感落点 + 记忆点）
```

## Thousand-Character Outline

Use for micro-films and standard films.

```text
第一幕（约 25%）：____（含 3-5 个关键场景节点）
第二幕（约 50%）：____（含 5-8 个关键场景节点）
第三幕（约 25%）：____（含 3-5 个关键场景节点）

每个场景节点须含：
   · 场景定位（地点 + 时间）
   · 主要事件
   · 情感 / 信息转折点
```
